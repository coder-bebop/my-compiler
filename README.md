# my-compiler

Operaciones permitidas:

    Aritméticas:
        Suma +
        Resta -
        Multiplicación *
        División /
        Exponenciación ^
    Comparación:
        ==
        != 
        >
        <
        >=
        <=
    Booleanas
        and 
        or

Un sistema de tipos:

    Int
    Float
    String
    Bolean
    

Flujos de control existentes:

    If, else, elif
    while
    do while
    for
